Подключение к экрану
  1. Для проводного подключения на столе вытяните провод и подключите
    к своему устройству. Убедитесь, что ваше устройство перешло в режим показа.
  2. Чтобы убрать провод обратно, нажмите кнопку в лючке около провода.
  3. Для вывода на экран изображения с компьютера Сигма нажмите кнопку
    «ПК».
  4. Для просмотра каналов телевидения нажмите кнопку «ТВ». Кнопками и
    можно переключать каналы. Кнопка открывает меню выбора
    частоиспользуемых каналов.
  5. Для подключения к видеоконференции нажмите кнопку ВКС. Откроетсяменю
    набора номера видеоконференции.
Телевидение
  1. Нажмите кнопку нужного канала телевидения, чтобы увидеть его
    наэкране.
  2. Для возврата в предыдущее меню нажмите кнопку в левой части
    панели.
Набор номера ВКС
  1. Наберите номер видеоконференции (четырёхзначный номер
    виртуальной комнаты), нажмите кнопку «Начать вызов».
  2. Для возврата в предыдущее меню (без установления связи) нажмите
    кнопку в левой части панели.
ВКС
  1. После подключения к видеоконференции не забудьте включить
    микрофон, если это необходимо.
  2. Кнопкой «Тип экрана» можно изменять расположение окон
    видеоконференции на экране телевизора.
  3. Для показа презентации от источников всем участникам
    видеоконференции нажмите нужную кнопку снизу. Устройство должно
    быть заранее подключено к системе (беспроводным или проводным
    способом).
  4. Для остановки показа презентации ещё раз нажмите выбранную кнопку.
  5. Чтобы отключиться от видеоконференции нажмите кнопку «Завершить
    вызов». Появится меню подтверждения отключения.
